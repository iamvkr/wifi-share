
# Wifi Share

### Frontend

Transfer files and Messages over local wifi network without internet. Made with react js

## Tech Stack

- React js
- Tailwind css
- Shadcn
- Socket io

## Run

Navigate to the project

```bash
  cd Frontend
```

Install dependencies

```bash
  npm install
```

Note:set the configuration in  `conf/conf.js`

Start the server

```bash
  npm run dev
```

build for production

```bash
  npm run build
```
